# STM32-MCU-Using-Embedded-Tools

An embedded systems project repository focused on developing and testing applications using STM32 Microcontrollers with industry-standard embedded development tools.  
This repository currently demonstrates LED blinking applications, GPIO configuration, and basic firmware development concepts using STM32 boards.

---

## 📌 Project Overview

This project is designed for beginners and embedded systems enthusiasts who want to learn:

- STM32 Microcontroller programming
- Embedded C development
- GPIO peripheral configuration
- LED interfacing and control
- Microcontroller debugging
- Firmware development workflow

The repository includes:
- Embedded source code files
- STM32 project configurations
- Hardware setup images
- Demonstration of blinking LED applications

---

## 🛠 Technologies & Tools Used

### Hardware
- STM32 Development Board
- On-board LEDs
- USB Programmer / ST-Link

### Software & IDEs
- STM32CubeIDE
- Embedded C
- STM32 HAL Libraries
- Keil uVision *(optional)*
- STM32CubeMX *(optional for configuration)*
